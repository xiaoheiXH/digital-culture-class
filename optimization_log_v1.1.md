  - 我的收藏：替换为 `favorites.svg`

  - 设置：替换为 `settings.svg`

## 2. 核心 Bug 修复与体验优化

### 2.1 详情页滚动条修复
- **问题描述**：在 v1.0 版本中，地点详情页（包含实景照片和评论列表）内容过长时，页面无法滚动，导致底部内容被截断。
- **解决方案**：重构了 `.detail-view` 和 `.detail-body` 的 CSS 布局。引入了 `flex-direction: column` 结合 `flex: 1` 和 `overflow-y: auto`，确保详情头部固定，内容区域支持原生丝滑滚动。

### 2.2 实景图片与图标全屏查看功能
- **问题描述**：用户反馈详情页中的缩略图太小，无法点击放大查看细节。
- **解决方案**：
  - 新增了一个 `id="imageViewer"` 的全局全屏图片查看器（Modal），支持半透明黑色背景遮罩。
  - 在 `app.js` 中为地点的“封面图标”和“实景照片列表”增加了点击事件监听 (`onclick` & `addEventListener`)。
  - 用户现在可以点击任意图片，在全屏模式下无损查看，点击背景或关闭按钮即可退出全屏。

## 3. 账号系统与社交功能重构

### 3.1 自主注册功能
- **账号系统升级**：重构了原本单一的“硬编码账号”登录界面。现在在登录面板中新增了“登录/注册”双 Tab 切换功能。
- **自主注册**：用户只需输入自定义的账号名和密码即可直接完成注册并自动登录，大大降低了系统门槛。用户数据被存储在本地的 `usersDB` 中（作为原型展示）。

### 3.2 标点收藏功能
- **收藏按钮**：在地点详情页头部新增了“⭐ 收藏”按钮，点击即可高亮（变黄）并保存至个人收藏夹。
- **我的收藏**：在“我的”页面新增了“我的收藏”入口，点击弹出列表，再次点击列表中的地点可自动跳转回地图并定位到该地点。

### 3.3 评论聚合与“我的评论”
- **我的评论**：在“我的”页面新增了“我的评论”入口，点击可统一查看当前用户在整个地图中留下的所有足迹和评论，点击可快速跳转回对应地点。

### 3.4 好友系统与标点转发
- **好友管理**：在“我的”页面新增了“我的好友”入口。用户可以在弹窗中输入账号名（需对方已注册）并添加为好友。
- **标点分享/转发**：
  - 在地点详情页头部新增了“📤 转发/分享”按钮。
  - **如果已添加好友**：点击转发会弹出一个列表供你选择好友并一键发送。
  - **如果没有好友或未登录**：点击转发会自动将包含地点 ID 的链接复制到剪贴板，方便通过微信等外部工具分享。

### 3.5 详情页点赞功能
- **点赞互动**：在地点详情页的内容描述下方新增了“👍 点赞”按钮。
- **点赞限制**：实现“每人限点一次”逻辑（支持取消点赞）。对于注册用户通过其 `username` 记录，对于匿名访客通过生成本地 UUID `guest_id` 来唯一标识其设备身份，防止重复刷赞。

### 3.6 匿名评论逻辑调整
- **强制匿名**：在之前的版本中，匿名访客可以自由填写昵称。为规范管理，现已隐藏访客的昵称输入框，并在代码逻辑层强制所有未登录访客的评论均以“匿名用户”的身份展示。
